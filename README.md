# Generative Models for Therapeutic Discovery

Quarto website for the workshop **Generative Models for Therapeutic Discovery: Learning molecules, targets and cell-state responses**.

## Render locally

```bash
quarto render
```

The rendered website is written to the `docs/` folder.

## GitHub Pages

Publish from branch `main`, folder `/docs`.

## Organiser photos

The organiser cards currently use placeholder SVG files in:

```text
assets/img/organisers/
```

To add real photos, replace the placeholders or add new images, for example:

```text
assets/img/organisers/dario-righelli.jpg
assets/img/organisers/cristian-taccioli.jpg
assets/img/organisers/pietro-lio.jpg
```

Then update the corresponding image paths in `organisers.qmd`.
Square images work best because the CSS crops them into circles.
